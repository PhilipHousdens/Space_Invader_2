# Space Invader 2

A classic Space Invaders game built with JavaFX.

## Description

This is a 2D space shooter game where you control a spaceship to defend against waves of alien invaders. Features multiple enemy types, a boss battle, and immersive sound effects.

## Features

- Classic arcade-style gameplay
- Multiple enemy types (common and uncommon enemies)
- Boss fight
- Sound effects and background music
- Score tracking
- Lives system

## Requirements

- Java 20
- Maven 3.6+

## How to Run

1. Clone the repository
2. Navigate to the project directory
3. Run `mvn clean javafx:run`

## Controls

- Arrow Left/Right: Move the spaceship
- Spacebar: Shoot regular bullets
- R: Shoot special rocket bullets

## Gameplay

- Defeat all enemies to spawn the boss
- Defeat the boss to win
- Avoid enemy bullets and collect points by destroying enemies
- You have 3 lives
